# Plugin Images

This directory contains images used within the plugin admin interface.

## Files

- `logo.png` - Plugin logo displayed in admin footer (recommended: 200x50px transparent PNG)

## Usage

The logo is referenced in:
- `includes/admin-page.php` - Alt Text Manager page footer
- `includes/audit-dashboard-page.php` - Audit Dashboard page footer

Replace `logo.png` with your actual logo file maintaining the same filename.
