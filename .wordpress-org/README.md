# WordPress.org Assets

This directory contains assets for the WordPress.org plugin directory listing.

## Required Assets

When submitting to WordPress.org, upload these files to the SVN `assets/` directory (separate from plugin code):

### Plugin Icons
- `icon-256x256.png` - Square icon (256x256px)
- `icon-512x512.png` - Square icon high-DPI (512x512px)

### Plugin Banners
- `banner-772x250.png` - Header banner (772x250px)
- `banner-1544x500.png` - Header banner high-DPI (1544x500px)

### Screenshots
- `screenshot-1.png` - Media Library with inline alt-text editing
- `screenshot-2.png` - Audit Dashboard overview with statistics
- `screenshot-3.png` - Missing Alt-Text report with quick-edit
- `screenshot-4.png` - User attribution breakdown
- `screenshot-5.png` - Advanced filtering interface
- `screenshot-6.png` - Scans tab with HTML reports

## Asset Guidelines

- **Icons**: Square, clear logo/symbol
- **Banners**: Landscape format, professional design
- **Screenshots**: Actual plugin screenshots showing features
- All images should be optimized for web (use PNG format)

## SVN Upload

These assets go in the separate `assets/` directory in SVN:
```
https://plugins.svn.wordpress.org/wp-alttext-auditor/assets/
```

NOT in the trunk or tags directories!
